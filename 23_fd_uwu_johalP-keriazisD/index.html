Team uWu's Foundation Demo

Welcome to our Foundation demo! Here we will demonstrate some of the useful capabilities of the Foundation front-end framework. The easiest way to test out these features for yourself is to copy the following CDN link into the head of your HTML file:


<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">

Typography

Foundation gives us access to many cool new features to customize text on webpages. The following are a few examples:

Small Header Segments

You can enclose header text in a <small> tag to have it appear inline with the header, but in a smaller font with a lighter color.

Foundation Demo by Team uWu

Dividers

You can use <hr> to indicate a thematic break between paragraphs.


Definition Lists

These are used to display name-value pairs. A <dl> tag is used to start the list, terms are enclosed in <dt> tags, and definitions are enclosed in <dd> tags. In fact, this list is a defintion list!

Blockquotes

These are used to format quotes. Quotes are enclosed in <blockquote> tags, and within those tags, the source is encloced in <cite> tags.

No hablo queso. Topher Brown Mykolyk
Abbreviations

You can enclose text in <abbr> tags to have it display the full term when you hover over it with your mouse. The title attribute of this tag must be set equal to the term you want displayed upon hovering.

Code

You can display code by enclosing it in <code> tags. In order to display opening and closeing arrow brackets properly, you must use &lt; for opening brakets and &gt; for closing brackets.

Navbar

Adding a navbar is another cool feature of Foundation, though not much can be done without the use of javascript. We have a simple navbar at the top of this page. We were able to create this by using the <ul> tag and setting its class attribute to "menu". Each navbar option is a link enclosed in a list element. The home option is highlighted because the class attribute of its <li> tag is set to "is-active".
